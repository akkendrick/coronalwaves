<comment>
This file is a coronal shock event nugget template
It contains all the necessary information needed to obtain and analyze event data
This comment will be completely ignored by the parser
Here is a legend for the different types of info
<designation>:<edesignation> - event designation, for example, e05
<st>:<est> - start time of the event, like 2011/01/01 00:00:00
<et>:<eet> - start time of the event, like 2011/01/01 00:01:00 (non-inclusive in the minutes)
<coordx>:<ecoordx> - heliographic X coordinate in arcsec (can be negative)
<coordy>:<ecoordy> - heliographic Y coordinate in arcsec (can be negative)
<flareclass>:<eflareclass> - flare X-ray class from GOES observations
<typeII>:<etypeII> - radio type II burst: 0 for none, 1 for just fundamental, 2 for fundamental and harmonic
<typeIII>:<etypeIII> - radio type III burst: 0 for none, 1 for yes
<path>:<epath> - the location on ARIEN where the event files live

The event nugget file must be saved in the respective event's folder, for example
/Volumes/PLUME/AIA_data/studies/2011events/e05/
